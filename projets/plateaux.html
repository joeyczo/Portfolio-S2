Présentation

Plateaux est une plateforme de jeu en ligne créée dans le cadre de la SAE 2.3, installation de services réseaux. Nous avons créé ce jeu avec des technologies web, comme Node.js, TypeScript, HTML et CSS. Sur cette plateforme, on retrouve 4 jeux connus : le Uno, le Simon, le Memory et des questions de culture générale. Nous avons tous collaboré sur le projet et appris les uns des autres.
